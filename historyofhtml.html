People/History of HTML

Sir Tim Berners-Lee



             Sir Tim Berners-Lee is a British computer scientist who is credited as the inventor of the world wide web. He made the first concept of HTML and version 1.0 and 2.0.He was awarded knighthood by Queen Elizabeth II in 2004. He wrote the software for the first web server and the first web client in 1990-1991.

Dave Reggett



             Dave Ragget is a british computer specialist who worked on the world wide web. He also worked on HTML 3.0-4.0. HTML 5, the current version, was made by his organization W3C. (World Wide Web Consortium)

























Timeline of HTML

Birth of HTML (1989-1991): HTML was invented by Sir Tim Berners-Lee, a British computer scientist, in the late 1980s while working at CERN. In 1989, he developed the first proposal for what would become the World Wide Web. Initially, HTML was a simple markup language used to create static web pages with links.
HTML 1.0 (1991): The first version of HTML, HTML 1.0, was published in 1991. It included basic elements for structuring a web page such as headings, paragraphs, lists, and links.
HTML 2.0 (1995): HTML 2.0 introduced more features including tables, text formatting, and forms. This version laid the foundation for the web to become more interactive.
HTML 3.2 (1997): HTML 3.2 brought further enhancements such as support for scripts and applets, which allowed for more dynamic content on web pages. It also introduced support for style sheets.
HTML 4.01 (1999): HTML 4.01 was a major update that introduced features like cascading style sheets (CSS), which separated the structure and presentation of web pages. It also introduced support for multimedia elements like images and video.
XHTML (2000): XHTML (Extensible Hypertext Markup Language) was introduced as a reformulation of HTML 4.01 in XML syntax. XHTML aimed to make HTML more extensible and interoperable with other XML-based technologies.
HTML5 (2014): HTML5 is the latest version of HTML and represents a significant evolution of the language. It introduced many new features including native support for video and audio, canvas for drawing graphics, geolocation, local storage, and semantic elements such as <header>, <footer>, <nav>, etc. HTML5 also provides better support for mobile devices and accessibility.
