Intro to CSS tags

To add a little bit of sppice to your webpage, you will need to add some CSS. Here are some basic CSS tags you can add to your webpage:

Color: This property sets the color of text content. You can use color names (e.g., "red"), hexadecimal codes (e.g., "#FF0000"), RGB values (e.g., "rgb(255, 0, 0)"), or HSL values (e.g., "hsl(0, 100%, 50%)") to specify colors.

Font-Family: This property defines the font to be used for text content. You can specify multiple fonts as fallback options to ensure compatibility across different devices and platforms.

Font-Size: This property sets the size of the font. You can specify the font size using various units such as pixels (px), ems (em), or percentages (%).

font-weight: This property sets the thickness of the font. Common values include "normal" and "bold", but you can also use numeric values (e.g., 400 for normal, 700 for bold) or keywords like "lighter" and "bolder".

Text-Align: This property aligns text content within its container. You can align text to the left, right, center, or justify it (aligning both left and right edges).

Margin: This property sets the outer margin of an element, creating space around it. You can specify margin values for individual sides (top, right, bottom, left) or use shorthand notation to set them all at once.

Padding: This property sets the inner padding of an element, creating space inside it. Similar to margin, you can specify padding values for individual sides or use shorthand notation.

Background-color: This property sets the background color of an element. You can use color names, hexadecimal codes, RGB values, or HSL values to specify the background color.

Border: This property sets the border of an element. You can specify the border width, style, and color using shorthand notation or individual properties.

Display: This property specifies the display behavior of an element. Common values include "block" (which generates a block-level element), "inline" (which generates an inline element), and "inline-block" (which generates an inline-level block container).

Just like the HTML tags page, you can look at the CSS of each element! Just hit ctrl + shift + i and select an element.
