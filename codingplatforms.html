Coding Platforms


1.VSCode


VSCode is a platform where you can easily connect a file, and there are extensions that you can download to help you make your coding experience better.

2. Notepad ++


Notepad ++ is a text and source code editor used with Microsoft Windows. It allows working with multiple files in a single window.

3. Replit


Replit is a coding platform with many opportunities. There are tutorial videos for beginners, bounties where you can code something that somebody else wants for money, and more.

Conclusion


These are only some of the coding platforms. There are many more out there for you to use. Now it's your turn. From here you decide what you want to use. Hope you have a great coding experience!
