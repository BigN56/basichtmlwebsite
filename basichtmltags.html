Intro to tags

When you type your tags, you should type 2 tags. 1 is an opening tag, and one is a closed tag. the opening tag should be surrounded by these 2 characters: < >. Your closed character should be surrounded by these characters: < /> There are some acceptions, like br. Br doesn't need a closed tag. You will soon learn what br is.

Basic HTML Tags

<!DOCTYPE html>

This tag is to tell your browser what version of HTML you are using. You want to use this tag at the start of all your code.

<html>

This tag represents the root for all your HTML code. You want this tag under DOCTYPE and around your HTML code.

<head>

This tag is for defining your document, like giving it a title. You want this tag in between HTML.

<body>

This tag is for all of the contents in your document. You would want this tag under head.

<title>

This tag is to name your document. You want this tag to be in between the head tag.

</br>

This tag is to leave a row empty with nothing in it. You want this tag in between the body tag wherever you want a row empty.

<img>

This tag is to add any image in your document. The current coding platform you are using should have access to a folder created by you. You would then put your images in that folder so that it will show up. You want this tag in between the body tag wherever you want to add an image.

<p>

This tag is for a paragraph. You can type any text using this tag. You want this tag in between the body tag wherever you want to add text.

Adding On To Opening Tags

There are many more tags than this. If you are wondering how this webpage was made, hit ctrl + shift + i. This will open the inspect window, which will show you how a webpage was made.

Basic layout of a webpage

A simple webpage might look something like this:

<!DOCTYPE html>
<html lang="en">
<head>
<title>Document Name</title>
</head>
<body>
<p>Some Text!</p>
</body>
</html>
