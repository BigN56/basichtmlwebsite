Intro to tags


When you type your tags, you should type 2 tags. 1 is an opening tag, and one is a closed tag. the opening tag should be surrounded by these 2 characters:< >. Your closed character should be surrounded by these characters:< /> There are some acceptions, like br. Br doesn't need a closed tag. You will soon learn what br is.

Adding On To Opening Tags



You can put words behind the opening tag to add style to your output. For example, you can change the size of your images using width and height, change the size and color of your font using size and color, and much more. This will be located in your opening tag. For example, if you want to change the size of your font, it will sort of look like this: < font size="+3">.
